# Свадебное приглашение Дмитрия и Яны

Статический сайт-приглашение в стиле медицинской карты. Его можно открыть как обычный `index.html` или выложить на любой хостинг для статических сайтов.

## Что уже готово

- Новый дизайн по референсу: медицинская карта, табличные строки, штамп, рукописные бордовые акценты.
- Добавлены фотографии пары из папки проекта.
- Добавлена локальная фотография главного корпуса ВГМУ для блока знакомства.
- Указаны дата `26.08.26`, время `14:00`, место `ЗАГС Рамонского района, улица Советская`.
- RSVP-форма подключена к Netlify Forms: после публикации на Netlify ответы сохраняются в панели сайта.
- Кнопка добавления события в календарь.
- Базовые настройки для публикации на Netlify, Vercel и GitHub Pages.

## Перед рассылкой

1. Открыть сайт с телефона и компьютера.
2. Проверить написание имен, даты, времени и адреса.
3. Проверить, что кнопка карты ведет туда, куда нужно.
4. После публикации на Netlify отправить тестовый RSVP и проверить, что ответ появился в разделе `Forms`.

## Как опубликовать на Netlify

1. Зайти на [netlify.com](https://www.netlify.com/).
2. Создать новый проект через `Add new site`.
3. Загрузить папку `wedding-invitation` вручную или подключить репозиторий.
4. В настройках публикации выбрать эту папку как корень сайта.
5. После публикации Netlify выдаст ссылку, которую можно отправлять гостям.

## Как проверить Netlify Forms

1. Открыть опубликованный сайт на Netlify.
2. Заполнить форму подтверждения присутствия.
3. Нажать `Отправить ответ`.
4. В панели Netlify открыть сайт, затем раздел `Forms`.
5. Открыть форму `wedding-rsvp`: там появится тестовый ответ.

## Как получать ответы на почту

1. В панели Netlify открыть опубликованный сайт.
2. Перейти в `Forms`.
3. Открыть форму `wedding-rsvp`.
4. Включить уведомления о новых ответах и указать нужную почту.

## Как опубликовать на Vercel

1. Зайти на [vercel.com](https://vercel.com/).
2. Создать новый проект и загрузить или подключить папку `wedding-invitation`.
3. Framework Preset оставить `Other`.
4. Build Command оставить пустым.
5. Output Directory оставить пустым или указать `.`.

## Как опубликовать на GitHub Pages

1. Создать репозиторий на GitHub.
2. Загрузить содержимое папки `wedding-invitation`.
3. В настройках репозитория открыть `Pages`.
4. Выбрать публикацию из ветки `main`, папка `/root`.
5. Дождаться появления публичной ссылки.
